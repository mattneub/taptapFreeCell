Import / Export

You can export or import a game as text. Export is performed via email. A typical scenario is that you export a game by emailing it to yourself or a friend, and the recipient copies the game out of the email and imports it to play it.

To access the Import / Export pane, tap the “ellipsis” button and choose Import / Export.

Export

Games are exported through email (which you might well send to yourself). The email contains two things:

  • The game’s initial deal, expressed in fc-solve format. This is useful for import, or for pasting into an online solver such as Shlomi Fish’s solver. You might do that, for example, if you are unsure whether the game is solvable.

  • The game’s moves, expressed in standard notation. Only moves that you made are included.

You can also construct this email by exporting a past game from the Statistics pane.

Import

If you import a game, you deal that game immediately and can begin playing it.

To import a game, paste its initial deal, expressed in fc-solve format, into the text view, and tap Import and Deal.
